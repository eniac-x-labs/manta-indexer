**Tips**: need [Go 1.18+](https://golang.org/dl/)

## Install

### Install dependencies
```bash
go mod tidy
```

Or check out the latest [release](https://github.com/eniac-x-labs/manta-indexer).

##  Test And Depoly

### Test
```
make test 
```

### Build

```
make 
```
